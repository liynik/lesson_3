# Инструкция для работы с markdown

## Выделение текста 

Чтобы выделить текст курсивом, необходимо обрамить его звездачками (*)
например *вот так*

Чтобы выделить текст полужирным, необходимо обрамить его жвойными звездачками (**) например **Вот так**
